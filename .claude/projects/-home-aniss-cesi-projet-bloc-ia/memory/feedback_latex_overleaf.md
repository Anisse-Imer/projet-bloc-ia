---
name: Sorties en LaTeX pour Overleaf
description: Toujours fournir les réponses de rédaction en .tex compatible Overleaf, pas en markdown
type: feedback
---

Lors de la rédaction des livrables, toujours fournir les réponses en format .tex que l'utilisateur peut copier-coller directement dans Overleaf.

**Why:** La professeure valorise grandement l'utilisation d'Overleaf et l'utilisateur travaille exclusivement avec cet outil pour les rapports.

**How to apply:** Pour toute tâche de rédaction de livrable, produire du code LaTeX complet et compilable. Ne pas donner de markdown quand il s'agit du rapport.
